## Answers to Questions;

### 1. What is the difference between getElementById, getElementsByClassName, and querySelector / querySelectorAll?

# Answer: 
getElementById is used to select a single element that has a specific ID. It returns only one element (or null if not found).
getElementsByClassName is used to select multiple elements that have a specific class name. It returns a live HTMLCollection of all matching elements.
querySelector is used to select the first element that matches a given CSS selector(such as class, id, or tag).
querySelectorAll is used to select all elements that match a given CSS selector. It returns all matching elements.
